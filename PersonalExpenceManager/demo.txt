🔐 Project Idea: Personal Expense Tracker API
🧠 Concept

Build an API where users can:

Register & log in
Manage their personal expenses
Categorize spending
View summaries
🧩 Core Features
1. 👤 Authentication
User registration (email + password)
Login with JWT authentication
Password hashing (e.g., bcrypt)
Protected routes (only logged-in users can access their data)
2. 💸 Expense CRUD

Each user can manage their own expenses:

Expense fields:

id
amount
category (Food, Travel, Bills, etc.)
description
date
owner_id (linked to user)

Endpoints:

POST /expenses → Create expense
GET /expenses → Get all user expenses
GET /expenses/{id} → Get one expense
PUT /expenses/{id} → Update expense
DELETE /expenses/{id} → Delete expense
3. 🗂️ Categories (Optional but useful)
Predefined or user-created categories
CRUD for categories
4. 📊 Bonus Features (to stand out)
Monthly expense summary
Filter by date/category
Pagination
Export data (CSV)
